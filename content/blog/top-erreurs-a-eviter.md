## Erreur n°4 : Négliger les accès (escaliers, parking)

### Le problème

**Vous remplissez le formulaire** :
- "Étage : 3e"
- Vous ne précisez pas : **sans ascenseur**

**Le déménageur chiffre** :
- Il suppose qu'il y a un ascenseur
- Devis : 1 000€

**Le jour J** :
- Déménageur voit qu'il n'y a **pas d'ascenseur**
- Portage de 30 m³ sur 3 étages = +2h de travail
- Déménageur : "Il y a un supplément de 250€"

**Coût de l'erreur : 200-400€** + tension avec le déménageur

---

### Comment l'éviter

**Décrivez précisément vos accès** ✅

**Au départ** :
- Étage : 3e **sans ascenseur**
- Distance porte ↔ camion : ~20m
- Escalier : Largeur 90 cm, tournant

**À l'arrivée** :
- Étage : 1er **avec ascenseur**
- Parking camion : Place dédiée devant l'immeuble
- Contraintes : RAS

**+ Ajoutez des détails** (escaliers, entrée, parking)

**→ Le déménageur chiffre correctement dès le départ** ✅

## Erreur n°5 : Mal emballer (50% des casses)

### Le problème

**Emballage rapide/négligé** :
- Vaisselle dans cartons sans papier bulle
- Miroirs/cadres sans protection
- Objets lourds + objets légers dans le même carton
- Cartons trop lourds (> 20 kg)

**Le jour J** :
- Assiettes cassées : 80€ de vaisselle perdue
- Miroir cassé : 150€
- TV mal protégée : 600€
- **→ 830€ de casse évitable** ❌

---

### Comment l'éviter

**Règles d'emballage pro** ✅

**1. Vaisselle et objets fragiles** :
- Papier journal ou papier bulle entre chaque assiette
- Verres : 1 par 1, debout, calés avec papier
- Carton marqué : **"FRAGILE – HAUT"**

**2. Miroirs et cadres** :
- Papier bulle + carton renforcé (ou carton spécial miroir)
- Marquer : **"FRAGILE – VERRE"**

**3. Livres** :
- Petits cartons (max 15 kg)
- Livres à plat (pas debout)

**4. Vêtements** :
- Cartons standards ou sacs-poubelle (rapide)
- Vêtements sur cintres : housses dédiées

**5. Électronique** :
- Cartons d'origine si possible (TV, ordinateur)
- Sinon : papier bulle + carton renforcé

**→ Emballage soigné = 0 casse** ✅

**Si formule Confort** : Le déménageur emballe tout → 0 erreur d'emballage possible.

## Erreur n°7 : Ne pas souscrire d'assurance adaptée

### Le problème

**Assurance de base** (incluse) :
- 620€/m³ en général
- Pour un T3 de 30 m³ : couverture max **18 600€**

**Valeur réelle de vos biens** :
- Canapé : 1 500€
- Lit : 800€
- Table : 600€
- Électroménager : 2 500€
- TV + informatique : 2 000€
- Vaisselle + déco : 1 500€
- Vêtements : 3 000€
- **→ Total : 12 000€+**

**Si tout est cassé** (incendie camion, accident) :
- Assurance de base couvre : 18 600€ ✅ (OK dans ce cas)

**Mais si valeur > 18 600€** (ex: maison avec meubles design) :
- **→ Sous-assuré** ❌

---

### Comment l'éviter

**Vérifiez le plafond d'assurance** ✅

1. **Estimez la valeur de vos biens** (rapide) :
   - Meubles : 5 000€
   - Électroménager : 3 000€
   - Électronique : 2 000€
   - Vêtements : 2 000€
   - Divers : 3 000€
   - **→ Total : 15 000€**

2. **Calculez la couverture** :
   - Assurance de base : 620€/m³ × 30 m³ = 18 600€
   - Votre valeur : 15 000€
   - **→ Couverture suffisante** ✅

3. **Si valeur > couverture** : Souscrivez assurance complémentaire
   - Coût : 50-150€
   - **→ Tranquillité garantie** ✅

- Vêtements que vous ne portez plus (50% de votre garde-robe)

- Volume : -20 à -30%

## Erreur n°10 : Mal choisir les cartons (effondrement garanti)

### Le problème

**Vous utilisez** :
- Vieux cartons récupérés (supermarchés)
- Déjà utilisés 2-3 fois → Affaiblis
- Tailles aléatoires (petit, moyen, géant)

**Le jour J** :
- Carton s'effondre → Vaisselle cassée : 100€
- Gros carton trop lourd → Impossible à porter → Déménageur refuse
- **→ Repacking en urgence + stress** ❌

---

### Comment l'éviter

**Cartons adaptés** ✅

**Tailles standards** :
- **Petit** (35×25×25 cm) : Livres, objets lourds (max 15 kg)
- **Standard** (50×35×35 cm) : Vêtements, objets cuisine, déco (max 20 kg)
- **Grand** (60×40×40 cm) : Oreillers, couettes, objets légers uniquement

**Qualité** :
- Cartons neufs ou 1ère utilisation
- Double cannelure si objets fragiles

**Où les trouver** :
- Gratuit : Supermarchés (demandez au response)
- Achat : Pack 20 cartons + scotch : 30-50€

**Scotch** : Large (48-50 mm) et résistant

## Erreur n°12 : Oublier de faire un état des lieux "déménagement"

### Le problème

**Vous déménagez**

**Vous oubliez de faire** :
- détails "avant déménagement" de votre ancien logement (murs, sols, plafonds)
- détails "après déménagement" (idem)

**2 semaines après** :
- Propriétaire retient 400€ de caution : "Vous avez abîmé le parquet"
- Vous : "Mais non, c'était déjà comme ça !"
- Propriétaire : "Prouvez-le"
- **→ Vous n'avez aucune preuve** ❌

**Coût de l'erreur : 200-800€** (caution retenue à tort)

---

### Comment l'éviter

**Faites des détails/vidéos "avant-après"** ✅

**La veille du déménagement** (ancien logement) :
- Vidéo complète de tout le logement (5 min)
- détails de chaque pièce (murs, sols, plafonds)
- détails des éventuels défauts pré-existants

**Le jour du déménagement** (nouveau logement) :
- Vidéo complète à l'arrivée (avant de déballer)
- détails de chaque pièce

**Le jour du départ** (ancien logement) :
- Vidéo complète après nettoyage
- détails de chaque pièce vide et propre

**→ Preuve incontestable = caution récupérée intégralement** ✅

## Erreur n°13 : Ne pas assurer le jour de déménagement

**Le jour J** :
- Vous transportez vous-même des objets de valeur en voiture (bijoux, ordinateur)

### Comment l'éviter

**Séparez "objets personnels précieux" et "reste"** ✅

**Transport déménageur** (camion) :
- Tout le reste (meubles, cartons, électroménager)
- **→ Couvert par l'assurance déménageur** ✅

## Erreur n°15 : Négliger le nettoyage (caution retenue)

### Le problème

**Vous déménagez en urgence**

**Vous nettoyez rapidement** :

**Nettoyage complet = caution récupérée** ✅

**Checklist nettoyage** :
- [ ] Sols : Aspirateur + serpillière (toutes les pièces)
- [ ] Murs : Lessive des traces (doigts, marques)
- [ ] Plafonds : Dépoussiérage (toiles d'araignée)

- [ ] Placards : Vides et propres
- [ ] Cave/garage : Balayés, vides

**Temps nécessaire** : 3-6h selon la taille

**Ou** : Société de nettoyage (150-300€) → Vous gagnez 5h
