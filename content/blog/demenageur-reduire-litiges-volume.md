## Pourquoi les estimations classiques échouent

### Méthode 1 : Estimation à l'œil (60-70% de précision)

**Process** :
- Client vous décrit au téléphone : "J'ai un T3, pas trop de meubles"
- Vous estimez à l'œil : "Disons 25 m³"

**Erreurs fréquentes** :
- Client minimise (consciemment ou non) : "Pas trop de meubles" → Réalité : Beaucoup
- Client oublie cave/garage : +8 m³
- Client ajoute des cartons de dernière minute : +3 m³
- **→ Écart : +11 m³ = +44% d'erreur** ❌

**Aucune preuve** : Votre parole contre la sienne.

**Taux de litige : 40-60%** 😱

---

### Méthode 2 : Visite technique (80-90% de précision)

**Process** :
- Vous vous déplacez chez le client (2-3h)
- Vous faites le tour du logement
- Vous notez (papier ou tablette) : "Salon : canapé, table, 4 cartons..."
- Vous estimez : "Je dirais 28 m³"

**Erreurs possibles** :
- **Placards fermés** : Vous ne voyez pas l'intérieur → Client dit "Je vais les vider" → Jour J : Pleins
- **Ajouts dernière minute** : Client récupère meubles chez ses parents (non vus lors de la visite)
- **Client n'a pas encore trié** : Vous voyez vaguement → Sous-estimation

**Écart : +5-8 m³ = +18-29% d'erreur** ⚠️

**Preuve limitée** : Notes manuscrites (contestables). Pas de détails systématiques.

**Taux de litige : 15-25%** ⚠️

---

### Méthode 3 : Formulaire manuel (70-80% de précision)

**Process** :
- Client remplit formulaire en ligne : "Combien de chaises ? Combien de cartons ?"
- Vous calculez volume à partir des réponses

**Erreurs fréquentes** :
- **Client oublie des objets** : Il ne pense pas à mentionner cave, garage, grenier
- **Client sous-estime les cartons** : Il dit "10 cartons" → Réalité : 30 cartons
- **Client ne connaît pas les volumes** : "1 armoire" → Réalité : Armoire 3 portes (2× plus volumineuse)

**Écart : +7-10 m³ = +25-40% d'erreur** ⚠️

**Aucune preuve visuelle** : Uniquement réponses textuelles (contestables).

**Taux de litige : 25-35%** ⚠️

## Les 3 clés pour éliminer les litiges

### Clé 1 : Précision de l'estimation (90-95%)

**Pourquoi c'est important** :
- Plus l'estimation est précise, moins il y a d'écart jour J
- Moins d'écart = moins de suppléments = moins de litiges

**Système Moverz : 85-90% de précision** (vs 80-90% visite technique, 60-70% à l'œil)

**Comment** :
- Client décrit TOUT (placards inclus dans l'inventaire)
- Base de données de 50 000+ déménagements
- Calcul standardisé (analyse cohérente)

**→ Écart moyen : ±3-4 m³ sur un T3 de 30 m³** (vs ±5-8 m³ pour visite humaine)

---

### Clé 2 : Preuve visuelle (détails HD)

**Pourquoi c'est important** :
- Sans preuve, c'est votre parole contre celle du client
- Avec inventaire détaillé, c'est **INCONTESTABLE**

**Ce que l'inventaire détaillé apporte** :
- **Liste complète** de chaque pièce
- **Détail** de chaque objet décrit
- **Horodatage** (date de prise de détails)
- **Inventaire exhaustif** : Si ce n'est pas sur les détails, ce n'était pas déclaré

**Exemple** :
- Client dit : "J'avais mentionné ma cave !"
- Vous montrez les détails : "Non, regardez. Aucune détail de cave. Vous avez validé 32 m³ sans cave."
- Client : "..." (coincé)

**→ détails = tribunal en votre faveur** ✅

---

### Clé 3 : Validation client

**Pourquoi c'est important** :
- Le client qui **valide** l'inventaire s'engage moralement et juridiquement
- Il ne peut plus dire "Je ne savais pas"

**Process Moverz Pro** :
1. Client remplit inventaire détaillé
2. Le système calcule volume : 32,5 m³
3. Client reçoit :
   - Volume : 32,5 m³
   - Inventaire détaillé pièce par pièce
   - Ses propres détails (il voit ce qu'il a déclaré)
4. **Client clique : "C'est exact"** → Validation
5. Email de confirmation envoyé : "Vous avez validé 32,5 m³ le [date]"

**→ Engagement client** ✅  
**→ Preuve de validation en cas de litige** ✅
