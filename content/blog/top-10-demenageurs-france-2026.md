## Les 10 critères pour évaluer un déménageur

### Critère 1 : Santé Financière (le + important)

**Pourquoi c'est crucial** :
- Un déménageur en difficulté = risque de faillite PENDANT votre déménagement
- Vos biens peuvent être saisis ou bloqués
- Acompte perdu si faillite avant le jour J

**Comment vérifier** :

#### A. Score Creditsafe (gratuit via Pappers)
```
Score 80-100 : Excellent (risque minimal)
Score 60-79 : Bon (risque faible)
Score 40-59 : Moyen (vigilance)
Score 20-39 : Faible (risque élevé) ❌
Score 0-19 : Très faible (à éviter absolument) ❌
```

**Où trouver** : [Pappers.fr](https://www.pappers.fr) → Chercher le nom du déménageur → Onglet "Santé financière"

#### B. Bilans comptables (derniers 3 ans)
**Signaux VERTS ✅** :
- Fonds propres positifs et en croissance
- Chiffre d'affaires stable ou en hausse
- Trésorerie > 3 mois de charges

**Signaux ROUGES ❌** :
- Fonds propres négatifs (dettes > actifs)
- Perte 2 années consécutives
- Retards de paiement déclarés

**Où trouver** : Pappers.fr → Onglet "Finances" → Télécharger les bilans PDF

---

### Critère 2 : Litiges Juridiques

**Pourquoi c'est crucial** :
- Un déménageur avec litiges récurrents = pattern de mauvaises pratiques
- Risque de conflit avec vous (surfacturation, casse non remboursée)

**Comment vérifier** :

#### Pappers Décisions (litiges tribunaux)
**Accès** : Pappers.fr → Onglet "Décisions de justice"

**Signaux ROUGES ❌** :
- 3+ litiges clients en 2 ans
- Condamnations pour pratiques commerciales trompeuses
- Procédure collective en cours

**Exemple** :
```
Société XYZ Transport
- 2024 : Condamnation tribunal (non-restitution acompte)
- 2023 : 2 litiges clients (casse non remboursée)
→ ÉVITER ❌
```

---

### Critère 3 : Avis Clients (avec analyse critique)

**Pourquoi les avis seuls ne suffisent pas** :
- Faux avis achetés (50 avis 5★ en 1 jour = suspect)
- Avis extrêmes (1★ ou 5★ seulement = manipulé)
- Peu d'avis récents (20 avis en 2020, 0 en 2025 = inactif ?)

**Comment analyser intelligemment** :

#### A. Note moyenne ET nombre d'avis
```
4,8/5 avec 3 avis = PEU FIABLE (trop peu de volume)
4,2/5 avec 247 avis = FIABLE ✅ (volume + constance)
3,8/5 avec 500 avis = ACCEPTABLE (honnête, mais vigilance)
3,0/5 ou moins = À ÉVITER ❌
```

#### B. Distribution des notes
**Pattern SAIN ✅** :
```
5★ : 60%
4★ : 25%
3★ : 10%
2★ : 3%
1★ : 2%
→ Courbe naturelle
```

**Pattern SUSPECT ❌** :
```
5★ : 85%
4★ : 2%
3★ : 1%
2★ : 2%
1★ : 10%
→ Probablement des faux avis 5★ + vrais mécontents 1★
```

#### C. Récurrence des plaintes
**Lisez les 20 derniers avis 1-2★** :

**Plaintes récurrentes = RED FLAG ❌** :
- "Surfacturation jour J" (répété 5+ fois)
- "Casse non remboursée" (répété 3+ fois)
- "Déménageurs alcoolisés" (même 1 fois = gravissime)

**Plaintes isolées = NORMAL** :
- 1 avis "retard de 2h" sur 200 avis = acceptable
- 1 avis "carton abîmé" sur 150 avis = normal

### Critère 4 : Assurances et Licences

**Pourquoi c'est crucial** :
- Sans assurance = vous n'êtes PAS couverts en cas de casse
- Sans licence = illégal (risque pénal)

**Comment vérifier** :

#### A. Assurance Responsabilité Civile Professionnelle (obligatoire)
**Demander** : "Pouvez-vous me fournir une attestation d'assurance RCP ?"

**Vérifier** :
- Assurance valide (pas expirée)
- Plafond de garantie : minimum 1 500€/m³ (pour 30 m³ = 45 000€)
- Nom de l'assureur (vérifier qu'il existe : Allianz, AXA, MMA, etc.)

**Si refus de fournir l'attestation = FUYEZ** ❌

#### B. Licence de transport (facultative mais rassurante)
**Vérifier** : Demander le numéro de licence de transport de marchandises

**Où vérifier** : [Registre des transporteurs](https://www.ecologie.gouv.fr) (recherche par SIREN)

**Bon à savoir** : Pas obligatoire pour déménagement, mais signe de sérieux si présent.

### Critère 5 : Ancienneté et Stabilité

**Pourquoi c'est important** :
- Entreprise < 2 ans = peu de recul (peut fermer rapidement)
- Entreprise 5-10 ans = stabilité prouvée

**Comment vérifier** :

**Pappers.fr** → "Date de création"

**Interprétation** :
```
< 1 an : Très récent (vigilance) ⚠️
1-3 ans : Récent (OK si bons avis + bilan sain)
3-10 ans : Établi ✅
10+ ans : Très établi ✅
```

**MAIS** : Une entreprise ancienne avec fonds propres négatifs = pire qu'une jeune entreprise saine !

**→ Ancienneté = bonus, mais pas critère principal**

**Pourquoi c'est crucial** :
- Tarifs opaques = risque de surfacturation jour J
- Devis vague = litiges garantis

- Devis détaillé (volume, accès, services, assurance, conditions)

- Conditions d'annulation claires

**Signaux ROUGES ❌** :
- "Devis sur place uniquement" (pression commerciale)
- "À partir de 500€" (= au moins 800€ en réalité)
- Refus de détailler les suppléments possibles

### Un déménageur peut-il faire faillite pendant mon déménagement ?

**Oui, ça arrive** :
- Déménageur encaisse l'acompte
- Fait faillite avant le jour J
- Vous perdez l'acompte + devez trouver un autre déménageur en urgence

**Comment se protéger** :

- ✅ Utiliser Moverz (exclusion automatique des déménageurs fragiles)

### Que faire si le déménageur casse un meuble ?

**Étapes** :
1. **Constat immédiat** : Signalez la casse AVANT de signer le bordereau de livraison
2. **Photos** : Prenez des photos du meuble cassé
3. **Déclaration écrite** : Notez sur le bordereau : "Canapé cassé pied gauche, estimation réparation 300€"
4. **Courrier recommandé** : Envoyez un courrier AR sous 5 jours avec photos + devis réparation
5. **Assurance** : Le déménageur transmet à son assureur
6. **Indemnisation** : Vous recevez un chèque (délai 30-60 jours)

**IMPORTANT** : Si vous signez le bordereau sans réserve, vous ne pourrez PLUS réclamer.

---

### Puis-je annuler un déménagement ?

**Oui, selon les conditions du contrat** :

**Standard marché** :
- 15 jours avant : Remboursement 100% acompte
- 7-14 jours avant : Remboursement 50% acompte
- < 7 jours : Acompte non remboursé

**Vérifiez TOUJOURS** les conditions d'annulation avant de signer.
