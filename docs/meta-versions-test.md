## ❌ À ÉVITER dans les métadonnées

- **"IA"** : Pas assez clair pour le grand public, peut faire peur
- **"48h"** : N'est pas dans le Hero, crée une attente
- **"Numéro masqué"** : Redondant avec "0 harcèlement"
- **Trop de détails techniques** : Google coupe à 155-160 caractères
